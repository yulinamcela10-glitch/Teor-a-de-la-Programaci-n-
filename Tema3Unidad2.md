# Descripción del problema
El siguiente programa cuenta cuántos números positivos ingresa el usuario, se supone que estos se leen positivos y se detiene el bucle cuando se lee un número negativo o cero
- Variables

Numero: Almacena cada número ingresado.

Contador: Cuenta cuántos números positivos se ingresaron.

- Bucle do while

El programa entra en el ciclo al menos una vez, porque en do while la condición se evalúa al final.

Dentro del ciclo:

Si numero > 0, aumenta el contador.

Luego pide al usuario un nuevo número.

El ciclo continúa mientras el número ingresado sea mayor que 0.

- Salida

Cuando el usuario ingresa un número 0 o negativo, el ciclo termina y el programa muestra:
